Welcome to Chat!

Chat is a simple chat application that runs on a python flask server, and currently has android and web clients.

Open Source

Chat is open source with BSD license on github. This means you can fork the code and customise it to suit your needs.

Portable

Along with sqlite, Chat's server is portable, with minimal setup required to get it up and running.

Dedicated rooms

Each Chat server has a list of rooms where people can talk about a certain topic.
